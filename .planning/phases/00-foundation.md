# Phase 00 - Foundation

## Goal

Create a Vercel-ready Next.js project with shared tracking, environment contract, and base UI.

## Tasks

- [ ] Add package/config files.
- [ ] Add app layout, global styles, and shared UI helpers.
- [ ] Add Supabase clients and environment validation.
- [ ] Add Spanish-only route shells for all PRD surfaces.
- [ ] Verify TypeScript and build scripts when dependencies are installed.

## Exit criteria

- `pnpm install` can install dependencies.
- `pnpm typecheck` and `pnpm build` are expected to run once dependencies are available.
