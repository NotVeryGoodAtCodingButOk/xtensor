# Phase 03 - Product surfaces

## Goal

Build the admin dashboard, factory iPad flow, and client portal required by the PRD.

## Tasks

- [ ] Build admin login and protected admin shell.
- [ ] Build Excel-like dashboard with filters and realtime hooks.
- [ ] Build machine, catalog, colors, workers, clients, shipped, and settings screens.
- [ ] Build factory unlock, worker selection, machine list, stage detail, and undo UI.
- [ ] Build tokenized client portal with buffered dates and realtime.

## Exit criteria

- Main PRD workflows are navigable.
- UI labels are Spanish-only.
- Mobile, iPad, and desktop layouts are checked.
